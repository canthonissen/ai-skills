# De Bono Thinking Tools — a skill suite

A set of skills that put the thinking methods from Edward de Bono's *de Bono's Thinking
Course* into practice. Thinking is treated as a deliberate, practical skill: each tool
directs your attention to a place you'd otherwise skip, one operation at a time, briefly.

The skills are **grouped by purpose** so the right set surfaces when the situation calls
for it.

## The six skills

| Skill | Purpose | Tools inside | Stage |
|-------|---------|--------------|-------|
| **debono-thinking** | Router & foundation. Start here for any "help me think this through." Explains the framework and sends you to the right toolkit. | TO·LO·PO·SO·GO framework, perception/patterns, deliberate-thinking attitude | all |
| **debono-explore** | Broaden perception *before* you judge or decide. | PMI, CAF, C&S, OPV | LO |
| **debono-create** | Generate alternatives and genuinely new ideas. | APC, Challenge, Concept Extraction, Random Input, Provocation (PO), Reversal | PO |
| **debono-decide** | Set the aim, weigh against values, and choose. | AGO, FIP, Emotions & Values check, decision procedure | TO + SO |
| **debono-act** | Turn the decision into action that gets done (operacy). | Objective → steps → first action → who/when → monitor → contingency | GO |
| **debono-invert** | The backward lens (Jacobi & Munger). Think from failure: bound the aim, surface hidden risks, crack stuck problems, pre-mortem the plan. | Anti-goal, Failure Inversion, Jacobi reframe, Pre-mortem, Via Negativa | all (esp. TO, LO, SO, GO) |

## How the book's 10 topics map to the skills

1. **The PMI** → debono-explore
2. **Alternatives** → debono-create (APC)
3. **Perceptions and Patterns** → debono-thinking (theory) + debono-create (why lateral thinking works)
4. **Lateral Thinking** → debono-create
5. **Information and Thinking** → debono-explore (CAF) + debono-thinking (the "what do I know/need?" habit)
6. **Other People** → debono-explore (OPV)
7. **Emotions and Values** → debono-decide
8. **Decision Making** → debono-decide
9. **Thinking and Doing** → debono-act (operacy)
10. **Deliberate Thinking** → debono-thinking

**Augmentation (beyond the original 10):** *Inversion thinking* — Carl Jacobi's "invert,
always invert" and Charlie Munger's discipline of avoiding failure → **debono-invert**. It
threads through the existing stages rather than sitting apart: the anti-goal at TO,
failure-inversion at LO (after PMI), the Jacobi reframe at PO (kept distinct from Reversal),
and the pre-mortem at SO/GO.

## How to use them

- **Don't know where to start?** Invoke **debono-thinking** and describe the situation —
  it routes you.
- **Know what you need?** Go straight to the toolkit: reacting to an idea → *explore*;
  stuck for ideas → *create*; choosing → *decide*; executing → *act*.
- **A full think-through** walks the spine: define the aim (TO), look (LO), generate
  possibilities (PO), narrow and choose (SO), act (GO).
- Keep each tool **brief (2–3 min)** and run **one at a time**. Look before you judge;
  generate before you evaluate; decide before you act.

## Installing

Each folder is a self-contained skill (a `SKILL.md` plus nothing else). Two ways to add
them:

1. **From the `.skill` files** (provided alongside this folder): open each one and use the
   **Save skill** button.
2. **Manually:** in the Claude desktop app, go to **Settings → Capabilities → Skills** and
   add each skill folder.

Install **debono-thinking** at minimum; add the five toolkits for the full set.

## A note on spirit

These are practical *attention-directing devices*, widely used in education and business —
not a scientific theory of the brain. They earn their keep by making you look where you
wouldn't have looked. Use them lightly and often.

*Source: Edward de Bono, "de Bono's Thinking Course" (BBC Books / Facts on File). The CoRT
tool names (PMI, CAF, C&S, AGO, FIP, APC, OPV) and lateral-thinking techniques are de
Bono's.*
